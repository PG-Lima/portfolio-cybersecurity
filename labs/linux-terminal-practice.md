# 🟧 Linux Terminal Practice  

## 🇺🇸 English Version

### Environment
- Linux distribution used: Debian (Virtual Machine)
- Access method: Local VM
- Objective: Strengthen command-line fundamentals

### Commands Practiced
- pwd → Displays current directory  
- ls → Lists files and directories  
- cd → Changes directory  
- mkdir → Creates directories  
- touch → Creates empty files  
- rm → Removes files  
- chmod → Modifies file permissions  
- clear → Clears terminal screen  

### Technical Insight
Working with the Linux terminal reinforces:
- Directory structure awareness  
- Permission control concepts  
- Command precision  
- System interaction fundamentals  

These fundamentals are essential in cybersecurity environments, since most security tools rely on Linux systems.

## 🇧🇷 Versão em Português

### Ambiente
- Distribuição Linux utilizada: Debian (Máquina Virtual)
- Método de acesso: VM local
- Objetivo: Fortalecer fundamentos de linha de comando

### Comandos Praticados
- pwd → Mostra o diretório atual

### Entendimento Técnico
Trabalhar com o terminal Linux reforça:
- Consciência da estrutura de diretórios  
- Conceitos de controle de permissões  
- Precisão em comandos  
- Fundamentos de interação com o sistema  

Esses fundamentos são essenciais na área de cibersegurança, já que a maioria das ferramentas de segurança utiliza ambientes Linux.
